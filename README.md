## Setup

postcss & autoprefixer px转vw实现响应式布局
集成three.js 3d地图；高德AMap地图；openlayers地图
$fetch请求方法的封装
lottie动画 & autoAnimate & d3 & three.js


```
**安装依赖
yarn && yarn prepare

运行
yarn dev

```

yarn build
